ЗАДАНИЕ

1. Написать тесты для функции utils/classNames, которая "склеивает" имена классов которые нужно добавить в аттрибут className. Тесты должны проверить, что
- можно передавать любое количество аргументов
- если аргументы не переданы - возвращается пустая строка
- если аргумент не строка и не объект - он игнорируется
- если аргумент это строка - она попадает в строку-результат
- если аргумент это объект - его ключ попадает в строку-результат, если его значени truthy

2. Написать тесты для компонента Button, которые проверяют, что
- дети компонента рендерятся как дети тега button
- при нажатии на кнопку вызывается функция onClick
- если kind не передан, то в className есть класс btn-default
- если kind="info", то в className есть класс btn-info
- если kind="danger", то в className есть класс btn-danger
- если isBig=true, то в className есть класс btn-big

3. Написть тесты для компонента SignUpForm, которые проверяют, что
- при вводе в поля формы их значение меняется
- при нажатии на кнопку Confirm вызывается функция onSignUp и туда передается объект со значениями из 3-х полей
- при нажатии на кнопку Cancel вызывается функция onCancel
- если пароль и его подтверждение не совпадают - появляется валидационное сообщение, а функция onSignUp не вызывается

4. Написать тесты на компонент Users, которые проверяют, что
- рендерится столько же пользователей в том же порядке, как и в prop-е users
- рядом с каждым пользователям есть 2 кнопки - Edit и Delete
- по нажатию на кнопку Edit вызывается функция Edit и туда передается id редактируемого пользователя
- по нажатию на кнопку Delete вызывается функция Delete и туда передается id удаляемого пользователя
